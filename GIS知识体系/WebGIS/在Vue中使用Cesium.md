# 在Vue中使用Cesium

## 直接使用官方提供的cesium包

- 创建vue3项目

- 下载包

```
npm install cesium --save
```

- 将cesium/Build/Cesium文件夹复制到public文件夹下

- 在index.html中引入js和css文件

```
<link rel="stylesheet" href="./Cesium/Widgets/widgets.css">
<script src="./Cesium/Cesium.js"></script>
```

实测可用，但是开发的时候没有语法提示
